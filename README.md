# Usage
<!-- usage -->
```sh-session
$ npm install -g wx-generator
$ wx-generator COMMAND

$ wx-generator (-v|--version|version)
wx-generator/1.0.0 darwin-x64 node-v10.13.0
$ wx-generator --help [COMMAND]
USAGE
  $ wx-generator COMMAND
...
```
